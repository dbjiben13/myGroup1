# 齐齐哈尔大学-计算机与控制工程学院
## 2013级 计算机科学与技术专业 数据库实训 第一小组作品 ---- 密码管理
本程序提供个人密码储存以及查询服务,不涉及联网,所有内容保存在本地数据库中.

##技术信息
Java编写<br>
跨平台,Swing界面,已在Linux(Kde)/Windows10/Mac OS X测试

##可供参考
本程序部分代码可作为标准范例<br>
Java SQLite数据库操作(增删改查/创建表/参数化查询)<br>
JTable 动态数据<br>
Java 剪贴板操作<br>
